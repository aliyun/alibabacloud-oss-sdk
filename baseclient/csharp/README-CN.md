## Alibaba Cloud OSS BaseClient for .Net


## 安装

Use .Net CLI ( Recommand )

    dotnet add package AlibabaCloud.SDK.OSS.BaseClient

Use Package Manager

    Install-Package AlibabaCloud.SDK.OSS.BaseClient
